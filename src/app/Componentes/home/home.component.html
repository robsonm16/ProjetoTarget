Para iniciarmos as configurações do banco são:

Altere a propriedade "Data Source" para a instancia de banco de dados SQL Server desejada.

A propriedade "Initial Catalog" Possui o nome do banco de dados escolhido por mim, caso deseje pode alterar, mas deve criar um banco com o mesmo nome configurado nesta propriedade.

Na janela "Console do Gerenciador de Pacotes execute o comando: update-database
